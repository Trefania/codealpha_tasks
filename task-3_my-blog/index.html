What I Learned from My Web Development Internship

My experience with codealpha internship in web development helped me learn important things:

  • I learned how to use HTML, CSS, and JavaScript to create websites like this blog.
  • I learned how to make websites look good on different devices and screens.
  • I got better at working with a team of developers and solving problems together.
  • I learned to make websites easy to use and understand for people.
  • I learned to write code that works well, is secure, and can be easily maintained.

This internship was a big step for me in becoming a good web developer. I'm excited to use these skills to make even better websites in the future.

Projects:

  • Temperature Converter Website

    • Developing a website that converts temperature recorded in one unit to another can be an excellent place for web developers to move forward in their web development journey. The measuring units of the temperature recorded in a particular unit can be converted with a temperature converter, necessitating you to build a dropdown menu with temperature scales..

  • Background Generator

    • A background generator is a great way to practice CSS skills and familiarize yourself with basic JavaScript concepts. In this project, you will select a basic or a gradient colour and generate it via code. You will then create a webpage that generates random background colours and allows users to customize and copy the generated colour code.

  • Landing Page

    • A landing page is one of the best web development projects for beginners. A foundational understanding of HTML and CSS is required. You will learn how to add columns, divide sections, arrange items, add headers, footers. Create an impressive page using your creativity. It is important to pay attention to the alignment, the padding, the color palette, and boxes.
